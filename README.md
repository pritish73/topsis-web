# TOPSIS Web Service (Python + Flask)

This project provides a web-based implementation of the TOPSIS (Technique for Order of Preference by Similarity to Ideal Solution) method.  
The system accepts a CSV file along with user-defined weights and impacts, computes TOPSIS ranking, and emails the result to the user as a CSV file.
